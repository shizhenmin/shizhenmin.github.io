## CentOS7

### 安装

```bash
yum -y install git
```



### 配置用户相关信息

```bash
git config --global user.name "shizhenmin"
git config --global user.email "shizhenmin@zju.edu.cn"
git config --global core.editor vim  #设置默认编辑器
```



### 初始化

```bash
mkdir devops
cd devops
git init

git init devops  #等效于上面的命令，同时创建目录和初始化git
```



### 添加新文件

```bash
echo "print('hello')" > hello.py
echo hello > hello.txt
git add .
```



### 查看暂存区

```plaintext
[root@server devops]# git status  #查看详细信息
# On branch master
#
# Initial commit
#
# Changes to be committed:
#   (use "git rm --cached <file>..." to unstage)
#
#       new file:   hello.py
#       new file:   hello.txt
#
[root@server devops]# git status -s  #查看简要信息
A  hello.py
A  hello.txt
```



### 确认到版本库

```bash
#把暂存库中的文件提交到版本库，-m 后面是对此次提交命名
git commit -m "add hello.txt"
```



### 添加跟踪文件

```bash
echo 'print("done.")' >> hello.py
git status -s
git commit -am "modify hello.py"  #同时做git add 和 git commit
```



### 删除跟踪文件

```bash
git ls-files  #查看版本库中的文件
git rm hello.txt
```
